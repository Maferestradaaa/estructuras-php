# Estructuras de Control y Arreglos en PHP

Mini sitio web **PHP** que demuestra condicionales (`if`, `elseif`, `else`, `switch`), bucles (`for`, `while`, `foreach`) y uso de **arreglos** (indexados, asociativos y multidimensionales). Incluye **componentes reutilizables** con `include()` para encabezado, menú y pie de página, además de un **contador de visitas** de sesión.

## Estructura
```text
/
├── index.php
├── condicionales.php
├── bucles.php
├── arreglos.php
├── css/
│   └── styles.css
├── includes/
│   ├── header.php
│   ├── menu.php
│   └── footer.php
└── img/
    └── .gitkeep
```

## Cómo ejecutar localmente
1. Requiere PHP 8+.
2. Clona o descarga este repositorio.
3. En la carpeta del proyecto, ejecuta:
   ```bash
   php -S localhost:8000
   ```
4. Abre `http://localhost:8000` en tu navegador.

## Publicación en 000webhost / InfinityFree
- Crea una cuenta y un **sitio**.
- Sube todos los archivos de la carpeta del proyecto al **document root** (normalmente `public_html/`).
- Asegúrate de mantener la misma estructura de carpetas.
- Verifica la URL pública: `https://tuusuario.000webhostapp.com` (o la que te asigne el servicio).

## GitHub
1. Crea un repositorio, por ejemplo `estructuras-php`.
2. Añade/Confirma tus archivos:
   ```bash
   git init
   git add .
   git commit -m "Proyecto: estructuras de control y arreglos en PHP"
   git branch -M main
   git remote add origin https://github.com/TU_USUARIO/estructuras-php.git
   git push -u origin main
   ```
3. Comparte el enlace público del repositorio.
